🏗️ سلة UA→GA4 Bridge — سلة بتبعت events بـ Universal Analytics format القديم.
الـ Bridge ده Custom HTML Tag في GTM بيحوّل الـ UA events لـ GA4 تلقائياً — من غير ما تغيّر في كود سلة.

طريقة الاستخدام:
1. انسخ الكود أو حمّله
2. في GTM → Tags → New → Custom HTML Tag
3. الـ Trigger: All Pages (يتشغّل قبل أي event tag)
4. التسلسل: في Tag Sequencing → Setup Tag → حطّه قبل كل الـ tags التانية
